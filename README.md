<h1 align="center">Fuc.js</h1>
<h3 align="center">MVVM响应式框架,用于学习vue原理</h3>

### Installing
```
  yarn
  yarn run dev  // 代码开发
  yarn run build  // 代码打包压缩
```

### Feature

* 优于操作Dom的数据驱动
* {{}}语法模板解析
* v-if 指令
